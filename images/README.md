# Product Images Directory

This directory contains product images organized by category for the Pastimes Second-Hand Fashion Store.

## Directory Structure

```
images/
├── jackets/          # Jackets and coats
├── pants/            # Jeans, trousers, shorts
├── shirts/           # T-shirts, polos, hoodies
├── shoes/            # Footwear
├── accessories/      # Sunglasses, bags, jewelry
└── README.md         # This file
```

## Image Guidelines

- **Format**: Use JPG or PNG files
- **Size**: Recommended 800x600 pixels for optimal display
- **File Naming**: Use descriptive names (e.g., `nike_jacket.jpg`, `levis_jeans.jpg`)
- **Quality**: Keep file sizes under 500KB for faster loading

## Adding New Images

1. Place images in the appropriate category folder
2. Use the image path in the product form (e.g., `images/shirts/new_product.jpg`)
3. Ensure images are web-optimized for best performance
